## 内容说明 (Content Description)
This repository includes:
*   **The modified source code of the Noname base game.** (The modifications are primarily to integrate the extension and may include fixes or adjustments.)
*   **The source code of the "Liuli Version" extension/mod.**

## 如何构建 (How to Build)
1.  Clone this repository.
2.  Follow the build instructions of the original Noname project.
3.  The Liuli extension is integrated into the `extension` directory.

## 关于分发包 (About the Distribution Package)
The downloadable distribution package (provided via other channels) is built from the source code in this repository. It is a complete, ready-to-run version for user convenience.
